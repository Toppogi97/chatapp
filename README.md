# Jonatans Chattapp

Ett simpelt chattprogram byggt med C# och Socket.IO.

## Kom igång

1. Klona git repository

git clone https://github.com/Toppogi97/chatapp.git 
cd chatapp

2. Kör igång appen

dotnet run

## Hur den används

1. Skriv ett användarnamn när du promptas.
2. Börja chatta i konsolen.
3. Skriv **"exit"** för att lämna/disconnecta från chatten.
